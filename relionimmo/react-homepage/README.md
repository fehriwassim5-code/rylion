# React Homepage (Kelibia)

## Fichiers
- `react-homepage/HomePage.jsx`
- `react-homepage/homepage.css`
- `react-homepage/App.jsx`

## Utilisation rapide
1. Copiez ces fichiers dans votre projet React.
2. Installez `d3` dans le projet React: `npm install d3`
3. Importez `App.jsx` ou `HomePage.jsx` dans votre point d'entree.
4. Verifiez que l'image existe dans `public/assets/img/kelibia/kelibia-beach.jpg`.

Vous pouvez changer l'image ici dans `HomePage.jsx`:
```jsx
backgroundImage: "url('/assets/img/kelibia/kelibia-beach.jpg')"
```
